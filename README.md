# 🎯 Objetivo

El objetivo de esta Evaluación Final Transversal fue diseñar una solución de Infraestructura como Código utilizando Terraform, desarrollando módulos reutilizables para los componentes de red, cómputo y almacenamiento.

Cada módulo fue documentado, versionado mediante Semantic Versioning y publicado en HCP Terraform Registry para validar su reutilización desde un proyecto principal, permitiendo desplegar infraestructura en AWS siguiendo buenas prácticas de modularidad, automatización y mantenibilidad.
